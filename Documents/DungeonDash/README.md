# DungeonDash

A fully procedural, asset-free roguelike built with MonoGame 3.8 and .NET 8. All graphics and sound effects are generated at runtime—no external assets required.

## Features
- Procedural dungeon generation (BSP rooms, corridors)
- Turn-based grid movement and combat
- Recursive shadowcasting fog of war
- Enemy AI (idle, patrol, chase)
- Items and pickups (health, attack, defense, gold)
- Leveling, stats, and XP system
- Custom runtime-generated 5x7 bitmap font
- Minimap and HUD
- Procedural sound effects (no audio files)
- Clean, modern C# codebase

## Controls
- **WASD / Arrow Keys**: Move
- **Tab**: Toggle minimap
- **R**: Restart game
- **N**: New dungeon
- **ESC**: Menu/Quit

## How to Run
1. Install [.NET 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) and [MonoGame 3.8](https://www.monogame.net/downloads/).
2. Clone this repo:
   ```
   git clone https://github.com/YOUR-USERNAME/DungeonDash.git
   cd DungeonDash
   ```
3. Build and run:
   ```
   dotnet run
   ```
